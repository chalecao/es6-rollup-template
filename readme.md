## 使用说明

### 初始化项目
```
$ git clone -o es6-rollup-template \
      -b master --single-branch \
      https://github.com/chalecao/es6-rollup-template.git \
      <your-project-name>
$ cd <your-project-name>
$ npm install

```

### 更新代码
```
$ git checkout master
$ git fetch es6-rollup-template
$ git merge es6-rollup-template/master
$ npm install
```
